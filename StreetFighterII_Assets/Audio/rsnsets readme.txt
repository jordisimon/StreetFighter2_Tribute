RSNSETS.DAT README:

     This .DAT file included can be used with my written utility, RENSET.  It's
meant to be CLRMAMEPRO compatible and works fine as such if used properly.
If you are having trouble with using RENSET, please use CLRMAMEPRO, using the
instructions included.  Future support of my RENSET will likely be stopped as
CLRMAME will do the job fine.

How to use this file in CLRMAME:

1) Place all the .RSN files into a folder named "RSNSETS"  (EG. c:\temp\RSNSETS)
2) Load the .DAT file into CLRMAME, pointing at the folder in which the RSNSETS
   folder resides, not the RSNSETS folder itself. (EG. c:\temp)
3) CLRMAME will assume this folder, RSNSETS, which is the default name in the .DAT
   file, is an archive in which all the individual  files are pieces of one
   big rom file (RSNSETS.ZIP) and do renames/deletes and additions as needed.
